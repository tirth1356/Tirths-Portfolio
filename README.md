## 🚀 About This Portfolio

A modern, responsive portfolio website showcasing my projects, skills, and achievements. Built with React, TypeScript, and Tailwind CSS, featuring smooth animations and a clean, professional design.

### ✨ Features

- 🎨 Modern, minimal design with dark mode
- 📱 Fully responsive for all devices
- ⚡ Fast loading and optimized performance
- 🎭 Smooth animations with Framer Motion
- 🎯 ATS-optimized content
- 🔒 No backend required (frontend-only)

## 🛠️ Tech Stack

- **Frontend:** React, TypeScript, Vite
- **Styling:** Tailwind CSS
- **Animations:** Framer Motion
- **Components:** Shadcn/ui
- **Icons:** Lucide React

## 📦 Installation

```bash
# Clone the repository
git clone https://github.com/tirth1356/portfolio.git

# Navigate to directory
cd portfolio

# Install dependencies
npm install
```

## 🏃 Running the Project

```bash
# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```
